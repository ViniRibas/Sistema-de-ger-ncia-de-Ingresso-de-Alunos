Sistema de controle de engressos de alunos da Faculdade Dom Bosco para o curso de direito com base na nota da OAB, sistema feito para gerenciamento em massa. Programado em PHP com arquitetura MVC.


Sistema de Gestão de Egressos - SiteV3
Este sistema foi desenvolvido para gerenciar alunos egressos, focado no controle de aprovação na OAB, gestão de documentos e comunicação eficiente por curso.

Funcionalidades e Requisitos Atendidos
Autenticação e Controle de Acesso
Login Administrativo: Acesso restrito via e-mail e senha.
Níveis de Permissão: Estrutura preparada para diferenciar administradores de usuários comuns.
Painel Completo: Menu lateral dinâmico com todas as funcionalidades para o perfil administrador.

Gestão de Alunos (Egressos)
Importação em Massa: Upload de arquivos Excel para cadastro ágil de alunos.
CRUD Completo: Inclusão manual, edição, visualização e exclusão de registros.
Listagem Detalhada: Exibição de Matrícula, Nome, E-mail, Telefone e Situação OAB.
Identificação Visual: Sinalização (ícone verde) para alunos aprovados na OAB.
Filtros Avançados: Busca por curso, nome ou status da OAB.

Gestão de Documentos
Armazenamento: Gerenciamento de documentos OAB vinculados ao aluno.
Download Direto: Links acessíveis para o administrador baixar os arquivos.
Rastreabilidade (Logs): Registro de todas as atividades de upload e download de documentos.
Comunicação e Exportação
Módulo de Egressos por Curso: Controle segmentado para envio de comunicados.
Envio de E-mails: Sistema de convites e avisos por turma.
Logs de Comunicação: Registro de quem enviou, quem recebeu e histórico de edições/exclusões.
Exportação de Dados: Geração de listas de e-mails em formato CSV ou Excel.

Relatórios e Estatísticas
Dashboard de Indicadores: - Total de alunos cadastrados.
Total de aprovados na OAB.
Distribuição de alunos por curso.
Identificação de pendências documentais.

Tecnologias Utilizadas
Linguagem: PHP
Banco de Dados: MySQL
Interface: HTML5, CSS3, JavaScript

Dependências: Vendor (incluindo bibliotecas para manipulação de Excel e PDFs)
